# 🌱 Crop Disease Detection Using Deep Learning

## 📌 Project Description

Crop Disease Detection is a Flask-based web application that uses a **Convolutional Neural Network (CNN)** to identify diseases in crop leaf images. Users can upload an image of a crop leaf, and the system predicts the disease with a confidence percentage. The application also provides treatment recommendations and prevention measures to help farmers detect diseases early and improve crop health.

---

## 🚀 Features

- Upload crop leaf images
- Detect 38 different crop diseases
- Display disease name
- Show confidence percentage
- Provide treatment recommendations
- Display prevention measures
- User-friendly web interface
- Real-time prediction using Deep Learning

---

## 🛠️ Tech Stack

---

## 🧠 Deep Learning Model

- Model: Convolutional Neural Network (CNN)
- Framework: TensorFlow & Keras
- Input Size: 224 × 224 RGB Images
- Output: 38 Crop Disease Classes

---

## 📂 Dataset

The model is trained on the PlantVillage Crop Disease Dataset containing images of healthy and diseased crop leaves.

---

## 📸 Project Workflow

1. User uploads a crop leaf image.
2. Image is preprocessed.
3. CNN model predicts the disease.
4. Disease name and confidence percentage are displayed.
5. Treatment and prevention recommendations are shown.

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/kaurkhushdeep140-afk/crop-disease-detection.git
```

Move into the project folder:

```bash
cd crop-disease-detection
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
